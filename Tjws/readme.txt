TJWS 1.114

Read release notes inside zip.

After unzipping :
Change to ./bin directory and start tjws.bat, tjws.sh, 
or tjws.lnk in dependency on your OS.
Access server at http://localhost, or http://localhost:8080 (on Unix/Linux/Mac)

Android and Blackberry versions are not bundled with this version.
Build them from source https://github.com/drogatkin/Atjeews
Or get from Google Play https://play.google.com/store/apps/details?id=rogatkin.mobile.web&hl=en
You can build .bar file using instruction on RIM web site.

Check TJWS home at http://tjws.sf.net and sources at https://github.com/drogatkin/TJWS2/tree/master/1.x
